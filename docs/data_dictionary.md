# Data Dictionary

This file documents every column used in analysis. Update it whenever a new field is introduced.

## How to Use

Each dataset gets its own section. For each column record:
- **Column name** — exact name as it appears in the file
- **Description** — what the field represents
- **Type** — data type (string, integer, float, date, etc.)
- **Source** — which dataset/file it comes from
- **Notes** — known quirks, nulls, encoding issues, or transformation applied
